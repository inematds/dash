Anatomia de um Card de Dados

O que e

Um Card de Dados e a unidade fundamental de informacao em dashboards. E um container visual que agrupa um valor principal, rotulo descritivo, indicador de tendencia e contexto adicional. Diferente de cards decorativos, cards de dados sao otimizados para leitura rapida e tomada de decisao.

Por que aprender

Cards bem projetados permitem que usuarios compreendam metricas em menos de 3 segundos. Sao o primeiro elemento que executivos olham ao abrir um dashboard. Um card mal projetado causa confusao e desconfianca nos dados.

KPIs - Key Performance Indicators

O que e

KPIs sao metricas criticas que medem o sucesso de um objetivo. Diferente de metricas comuns, KPIs estao sempre ligados a metas especificas e sao usados para tomada de decisao estrategica. Em dashboards, KPIs ocupam posicao de destaque e tem tratamento visual diferenciado.

Por que aprender

Saber distinguir KPIs de metricas secundarias e crucial. KPIs mal escolhidos levam a decisoes erradas. A visualizacao correta de KPIs permite identificar problemas antes que se tornem criticos.

Conceitos-chave

Tipos de KPIs em Dashboards:

1. RESULTADO (Lagging)
   - Receita, Lucro, Churn
   - Mostram o que JA aconteceu
   - Cor: neutro ou baseado em meta

2. DIRECIONADORES (Leading)
   - Leads, Conversoes, NPS
   - Preveem resultados futuros
   - Cor: destaque para atencao

3. OPERACIONAIS
   - Tempo de resposta, Uptime
   - Saude do sistema
   - Cor: verde/amarelo/vermelho

Formatacao de Numeros e Valores

O que e

Formatacao de numeros e a arte de exibir valores de forma legivel e apropriada ao contexto. Inclui abreviacao de grandes numeros (1.2M ao inves de 1.234.567), formatacao de moeda, percentuais, datas e numeros negativos. E crucial para a usabilidade do dashboard.

Por que aprender

Numeros mal formatados causam confusao e erros de interpretacao. "1234567" e muito mais dificil de ler que "R$ 1.23M". A formatacao correta reduz a carga cognitiva e acelera a tomada de decisao.

Conceitos-chave

Regras de Formatacao:

MOEDA
  âœ“ R$ 1.234,56    (valores pequenos)
  âœ“ R$ 1.2M        (milhoes)
  âœ“ R$ 3.4B        (bilhoes)
  âœ— R$ 1234567.89  (nunca!)

PERCENTUAIS
  âœ“ 12.5%          (uma casa decimal)
  âœ“ 0.05%          (valores pequenos)
  âœ— 12.456789%     (precisao excessiva)

GRANDES NUMEROS
  1.000      â†’ 1K
  1.000.000  â†’ 1M
  1.000.000.000 â†’ 1B

DATAS
  âœ“ 15 Jan 2024    (legivel)
  âœ“ Ontem, 14:30   (relativo)
  âœ— 2024-01-15T14:30:00Z (tecnico)

Exemplo Pratico

// Funcao de formatacao
function formatNumber(value, type = 'number') {
  if (type === 'currency') {
    if (value >= 1e9) return `R$ ${(value/1e9).toFixed(1)}B`;
    if (value >= 1e6) return `R$ ${(value/1e6).toFixed(1)}M`;
    if (value >= 1e3) return `R$ ${(value/1e3).toFixed(1)}K`;
    return new Intl.NumberFormat('pt-BR', {
      style: 'currency', currency: 'BRL'
    }).format(value);
  }

  if (type === 'percent') {
    return `${value.toFixed(1)}%`;
  }

  return new Intl.NumberFormat('pt-BR').format(value);
}

// Uso
formatNumber(1234567, 'currency') // "R$ 1.2M"
formatNumber(0.125, 'percent')     // "12.5%"

Indicadores de Tendencia

O que e

Indicadores de tendencia mostram a direcao e magnitude da mudanca de uma metrica ao longo do tempo. Podem ser setas, percentuais de variacao, sparklines ou comparacoes com periodos anteriores. Sao essenciais para contexto temporal.

Por que aprender

Um numero sozinho nao tem significado. "R$ 100.000" e bom ou ruim? Depende se subiu 50% ou caiu 30%. Tendencias transformam dados estaticos em insights acionaveis.

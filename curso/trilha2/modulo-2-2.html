Quando Usar Cada Tipo de Grafico

O que e

Cada tipo de grafico serve para um proposito especifico. Graficos de linha mostram tendencias, barras comparam categorias, pizza mostra proporcoes. Escolher errado distorce a mensagem dos dados.

Por que aprender

O grafico errado pode levar a interpretacoes equivocadas. Um grafico de pizza com 15 fatias e ilegivel. Um grafico de linha para dados categoricos nao faz sentido. A escolha correta comunica instantaneamente.

Chart.js - Setup e Configuracao

O que e

Chart.js e a biblioteca de graficos mais popular para web. Leve (~60KB), responsiva, animada e com 8 tipos de graficos nativos. Perfeita para dashboards por ser simples de configurar e altamente customizavel.

Por que aprender

E o padrao da industria para graficos em dashboards. Documentacao excelente, comunidade ativa e compativel com qualquer framework. Alternativas como ApexCharts ou Recharts seguem padroes similares.

Conceitos-chave

<!-- 1. Incluir Chart.js -->
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

<!-- 2. Canvas para o grafico -->
<canvas id="myChart" class="w-full h-64"></canvas>

<!-- 3. Configuracao -->
<script>
const ctx = document.getElementById('myChart');

new Chart(ctx, {
  type: 'line', // bar, pie, doughnut, radar...
  data: {
    labels: ['Jan', 'Fev', 'Mar', 'Abr', 'Mai', 'Jun'],
    datasets: [{
      label: 'Vendas 2024',
      data: [12, 19, 3, 5, 2, 3],
      borderColor: '#3b82f6',
      backgroundColor: 'rgba(59, 130, 246, 0.1)',
      tension: 0.4, // curva suave
      fill: true
    }]
  },
  options: {
    responsive: true,
    maintainAspectRatio: false,
    plugins: {
      legend: { display: false }
    }
  }
});
</script>

Graficos de Linha e Area

O que e

Graficos de linha mostram tendencias ao longo do tempo. Graficos de area sao linhas com preenchimento, uteis para mostrar volume ou acumulado. Ambos sao essenciais para dashboards de monitoramento.

Por que aprender

90% dos dashboards precisam mostrar "como algo mudou ao longo do tempo". Vendas mensais, usuarios ativos, temperatura - tudo e tendencia. Dominar graficos de linha e fundamental.

Graficos de Barra e Histogramas

O que e

Graficos de barra comparam valores entre categorias. Barras verticais para poucas categorias, horizontais para muitas ou labels longos. Histogramas mostram distribuicao de frequencia.

Por que aprender

Comparacao e essencial em dashboards. "Qual produto vende mais?", "Qual regiao performa melhor?". Barras respondem essas perguntas instantaneamente.

Conceitos-chave

// Grafico de Barras - Chart.js
new Chart(ctx, {
  type: 'bar',
  data: {
    labels: ['Produto A', 'Produto B', 'Produto C', 'Produto D'],
    datasets: [{
      label: 'Vendas',
      data: [120, 190, 80, 140],
      backgroundColor: [
        'rgba(59, 130, 246, 0.8)',  // azul
        'rgba(16, 185, 129, 0.8)', // verde
        'rgba(245, 158, 11, 0.8)', // amarelo
        'rgba(139, 92, 246, 0.8)'  // roxo
      ],
      borderRadius: 8
    }]
  },
  options: {
    indexAxis: 'y', // barras horizontais
    plugins: {
      legend: { display: false }
    },
    scales: {
      x: { beginAtZero: true } // SEMPRE comece em zero!
    }
  }
});

Graficos de Pizza e Donut

O que e

Graficos de pizza e donut mostram proporcoes de um todo. Donut e preferido por permitir colocar informacao no centro. Limitados a poucas categorias (max 5) para serem efetivos.

Por que aprender

Proporcoes sao frequentes: market share, distribuicao de vendas por categoria, alocacao de orcamento. Mas pizza e frequentemente mal usado - saber quando NAO usar e tao importante quanto saber usar.

Conceitos-chave

âœ“ 2-5 categorias apenas
âœ“ Uma fatia e claramente dominante
âœ“ Comparacao exata nao e necessaria

QUANDO NAO USAR:
âœ— Mais de 5 categorias
âœ— Fatias de tamanho similar
âœ— Comparacao precisa necessaria

Estilizacao e Temas para Graficos

O que e

Estilizar graficos significa adapta-los ao design system do dashboard: cores, fontes, bordas, sombras. Inclui configurar temas claros e escuros, tooltips customizados e animacoes.

Por que aprender

Graficos com estilo padrao quebram a harmonia visual. Um grafico branco num dashboard dark mode e jarring. Consistencia visual aumenta a confianca e profissionalismo.

Conceitos-chave

// Tema Dark Mode para Chart.js
Chart.defaults.color = '#9ca3af';           // texto
Chart.defaults.borderColor = '#374151';     // linhas grid
Chart.defaults.backgroundColor = '#1f2937'; // fundo

// Configuracao Global
Chart.defaults.font.family = 'Inter';
Chart.defaults.font.size = 12;
Chart.defaults.plugins.tooltip.backgroundColor = '#111827';
Chart.defaults.plugins.tooltip.titleColor = '#f3f4f6';
Chart.defaults.plugins.tooltip.bodyColor = '#9ca3af';
Chart.defaults.plugins.tooltip.borderColor = '#374151';
Chart.defaults.plugins.tooltip.borderWidth = 1;
Chart.defaults.plugins.tooltip.padding = 12;
Chart.defaults.plugins.tooltip.cornerRadius = 8;

// Paleta de Cores para Dashboards
